# Grade Calculator

A simple web-based grade calculator that helps students track their course assignments and calculate weighted grades.

## Features
- Add assignments with scores and weights
- Automatically calculates current grade
- Shows contribution of each assignment to final grade
- Delete assignments
- Clean, responsive interface

## How to Use
1. Open `index.html` in your browser
2. Enter assignment details (name, points earned, points possible, weight)
3. Click "Add Assignment"
4. View your current grade and individual contributions
